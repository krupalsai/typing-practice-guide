# 🖐️ Finger Placement Guide

## Home Row Position

```
Left Hand          Right Hand
  A  S  D  F         J  K  L  ;
  ↑  ↑  ↑  ↑         ↑  ↑  ↑  ↑
Pinky Ring Mid Index  Index Mid Ring Pinky
```

**Both thumbs rest on the SPACEBAR.**

## Posture Tips

- **Back:** Straight, slightly reclined
- **Arms:** Parallel to floor, elbows at 90°
- **Wrists:** Floating above keyboard (don't rest while typing)
- **Screen:** Eye level, 50–70 cm away
- **Feet:** Flat on floor

## Common Mistakes to Avoid

❌ Looking at the keyboard  
❌ Using wrong fingers for keys  
❌ Tensing up your shoulders  
❌ Typing too fast before accuracy is solid  
✅ Always return fingers to home row after each keypress
